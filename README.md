- How to run?

  - move to `react-testing-traversy` directory

    ```bash
    cd react-testing-traversy
    ```

  - install dependencies and run the app

    ```bash
    # install dependencies
    yarn install

    # run the app
    yarn dev
    ```

  - run the unit test

    ```bash
    yarn test
    ```

  - run the End-to-End test using cypress

    ```bash
    yarn run cypress open
    ```
